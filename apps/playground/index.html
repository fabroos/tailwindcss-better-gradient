vs Tailwind Default Gradients

Compare the natural easing curve of Better Gradient with Tailwind's linear gradients

Tailwind Default (Linear)

from-purple-500 to-transparent

❌ Linear opacity transition
❌ Abrupt at start and end
❌ Less natural looking

Better Gradient (Eased)

fade-purple-500 fade-to-b

✓ Ease-out cubic curve
✓ Smooth, natural transition
✓ Professional appearance

Usage Examples

// Basic fade from top

<div class="fade-blue-500 fade-to-t"></div>

// Fade with any Tailwind color

<div class="fade-rose-600 fade-to-b"></div>

// Diagonal fade

<div class="fade-purple-500 fade-to-br"></div>

// Control step count

<div class="fade-purple-500 fade-to-b fade-steps-12"></div>

// Combine with responsive utilities

<div class="fade-blue-500 md:fade-red-500 fade-to-t"></div>
